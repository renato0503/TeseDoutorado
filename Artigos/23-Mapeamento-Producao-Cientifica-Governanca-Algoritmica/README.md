## Como obter os dados

1. Acesse https://www.webofscience.com (acesso institucional Fucape)
2. Use a query no script `Scripts_Extracao/coletar_wos.py`
3. Exporte em formato "Tab delimited file" para `Raw_Data/wos_export.txt`
4. Execute o script para processar
