# Regras para Formatação de Tabelas Acadêmicas (ABNT + APA)

## Estrutura Geral
- Tabelas usam **apenas linhas horizontais**: uma no topo do cabeçalho, uma abaixo do cabeçalho, uma no final do corpo
- Fonte: 10pt, espaçamento simples
